## Se trata de um CRUD básico feito com TypeORM

- temos uma demonstração de testes.
- temos uma demonstração de workflow com Github Actions.

- Instalação
```bash
$ yarn install
```

- Rodando a aplicação
```bash
$ yarn start
```

- Testando as rotas
```bash
$ yarn test

# teste em monitoramento
$ yarn test:watch
```
